# [MYSQL] DATE_FORMAT

## 사용
`DATE_FORMAT(날짜 , 형식)` : 날짜를 지정한 형식으로 출력

## 형식

|구분기호|역할|
|---|---|
|%Y|4자리 연도|
|%y|2자리 연도|
|%M|긴 월(영문)|
|%b|짧은 월(영문)|
|%m|숫자 월(두자리)|
|%c|숫자 월(한자리는 한자리)|
|%d|일자(두자리)|
|%e|일자(한자리는 한자리)|
|%W|긴 요일 이름(영문)|
|%a|짧은 요일 이름(영문)|
|%H|시간 (24시간)|
|%I|시간 (12시간)|
|%i|분|
|%S|초|
|%r|hh:mm:ss AM, PM|
|%T|hh:mm:SS|

## 번외
* 연도 리턴 : `YEAR(date)`
* 시간 리턴 : `HOUR(date)`
